
# Annotated images subset

## Description

This folder contains a subset of seven of the annotated images. These are the images that are also displayed in supplementary figure 7 of the paper.
The images can be used to test the script ScoresForAnnotatedImages.R.

## Usage

Enter the path to the folder in the R script. The script will then automatically read in and analyse the 7 images.
